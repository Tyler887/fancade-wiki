## Input/Ouput

Sine outputs value between 1 and -1 depending on the given value to it which scales from rotation (0°–360°)

## Notes

Additionally , there are only 4 certain points where you can get a whole number :
- 0°/360° = 0
- 90° = 1
- 180° = 0
- 270° = -1
